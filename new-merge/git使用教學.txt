merge:
首先先切換到最終的branch (branch a)
可使用 git checkout <branch a>
再來，使用git merge <branch b>
這樣，branch b就會加入 branch a裡了


刪除遠端分支
git push <remote_name> -d <remote_branch_name>

branch
查看所有分支
git branch
查看所有分支(含遠端)
git branch -a
只查看遠端分支
git branch -r
分支同步
git fetch
拉取远程分支并同时创建对应的本地分支
git checkout -b <本地分支名x> <遠端分支>/<远程分支名x>
拉取所有分支代码
git pull --all

下載遠端分支
git checkout <分支>
